BANUB LTD static website prototype

Open index.html in a browser or deploy the entire folder to Netlify, Cloudflare Pages, GitHub Pages or another static host.

Working features:
- Responsive navigation
- Product filters and product pages
- Search modal
- Shopping cart saved in localStorage
- Quantity controls and cart removal
- Demo checkout form
- Contact and newsletter confirmation messages
- Legal, delivery, returns and FAQ pages

Before live launch:
1. Add the real business email and phone number.
2. Confirm actual products, prices, stock and suppliers.
3. Connect a real checkout/payment provider.
4. Replace draft legal text with reviewed policies that match actual operations.
5. Add a real returns address and delivery terms.
